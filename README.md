# Трекер прочитанных книг

Консольное приложение на Python для учёта прочитанных книг: добавления в коллекцию, просмотра списка, расчёта средней оценки и анализа статистики по авторам.

## Как запустить

1. Перейдите в папку с проектом в терминале.
2. Выполните команду:
```bash
python main.py
Пример работы
После запуска программы отображается главное меню:

=== Трекер прочитанных книг ===
1. Добавить книгу
2. Показать все книги
3. Средняя оценка
4. Статистика по авторам
5. Удалить книгу
6. Выход
Выберите действие: 
Примеры взаимодействия с меню:

Пункт 1 («Добавить книгу»): пользователь вводит автора, название книги и оценку (от 1 до 5). Программа проверяет, нет ли книги в списке — если есть, выводит сообщение «Эта книга уже есть в списке!». Если книги нет, добавляет её в коллекцию и сохраняет в books.json.

Пункт 2 («Показать все книги»): программа выводит список всех добавленных книг с указанием автора, названия, оценки и даты добавления.

Пункт 3 («Средняя оценка»): рассчитывается и выводится средняя оценка всех книг в коллекции (с округлением до двух знаков после запятой).

Пункт 4 («Статистика по авторам»): отображается количество книг для каждого автора в коллекции.

Пункт 5 («Удалить книгу»): пользователь вводит номер книги из списка — программа удаляет запись и обновляет books.json.

Пункт 6 («Выход»): программа завершается корректно.
